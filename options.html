About OpenTube SEO

OpenTube SEO is a 100% offline, open-source alternative to VidIQ.

All analysis runs in your browser — no data is ever sent to any server.

Features

  • SEO scoring with 18 heuristic checks
  • Keyword extraction and research
  • Competitor analysis and content gap detection
  • AI title/tag/description generation via WebLLM
  • YouTube Studio integration
  • Keyboard shortcuts
  • Data export/import
  • 100% free and open source

Technologies used

  • Ponytail — AI agent coding style (write only what's needed)
  • Graphify — knowledge graph architecture patterns
  • WebLLM — browser-based AI inference via WebGPU
